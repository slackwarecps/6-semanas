What is a good source of examples to include in a few-shot prompt?
[Resposta correta: Os pares de maior pontuação nas avaliações demonstram o output perfeito para o seu caso específico. Exemplos aleatórios (A) não têm qualidade garantida, os de menor nota (C) ensinam o padrão errado e exemplos genéricos (D) não refletem o caso de uso real.]

---
[ ] A - Generate random examples without going through any evaluation.
[ ] B - Use the input/output pairs with the highest score in the evaluations, since they show the ideal output for the case.
[ ] C - Use the pairs that got the lowest scores so the model learns from mistakes.
[ ] D - Copy generic examples from the documentation, unrelated to the use case.
