The web search and document analysis agents both investigate the same subtopics, doubling token usage without increasing coverage. What's the most effective way to address this?
O que a pergunta pede: most effective way to address a sobreposição de trabalho.

A pegadinha: O trabalho deveria ser dividido antes da delegação.

Raciocínio: A forma mais eficaz previne a sobreposição na origem: o coordenador divide o espaço antes de delegar. A é reativa e complexa (coordenação em tempo real); B desperdiça o paralelismo; D deixa o desperdício acontecer e só limpa depois. C evita o custo em vez de remediar.

---
[ ] A - Shared state where agents log their focus to avoid duplication dynamically.
[ ] B - Sequential execution: document analysis runs after web search, using its findings as context.
[ ] C - Have the coordinator partition the research space before delegation, assigning distinct subtopics/sources to each agent.
[ ] D - Let both finish in parallel, then have the coordinator deduplicate afterward.
