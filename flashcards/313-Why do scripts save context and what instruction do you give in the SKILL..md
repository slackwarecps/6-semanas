Why do scripts save context and what instruction do you give in the SKILL.md?
[Resposta correta: O script roda sem carregar o conteúdo no contexto, então instrua o Claude a rodá-lo, não a lê-lo (só a saída gasta tokens). A A inverte a instrução; a C ignora o custo da saída; a D contraria a economia de contexto.]

---
[ ] A - Because Claude reads the whole script into the context before running; instruct it to read it first
[ ] B - Because the script runs without loading its content into the context (only the output consumes tokens); instruct Claude to run it, not to read it
[ ] C - Because scripts do not consume tokens at any stage
[ ] D - Because the SKILL.md should contain all the script code inline
