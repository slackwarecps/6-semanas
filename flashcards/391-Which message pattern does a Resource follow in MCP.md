Which message pattern does a Resource follow in MCP?
[Resposta correta: Resources usam ReadResourceRequest com URI e ReadResourceResult com os dados. A é o padrão de tools. C descreve um modelo assíncrono que não é o dos resources. D é descoberta de tools, não leitura de resource.]

---
[ ] A - CallToolRequest with arguments and CallToolResult with the result.
[ ] B - Request-response: ReadResourceRequest with a URI and ReadResourceResult with the data.
[ ] C - Publishing to an asynchronous queue with no direct response.
[ ] D - ListToolsRequest followed by ListToolsResult.
