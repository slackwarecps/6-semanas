An agent has a systematic performance drop (78% vs 93%) whenever a specific term appears, even with good tool descriptions. What is the cause?
[Resposta correta: A queda sistemática ligada a um termo específico indica lógica explícita keyword-sensitive no system prompt. As descrições não são a causa quando o desempenho é bom sem o termo (A), o acerto na ausência do termo descarta limitação do modelo (B) e o padrão é sistemático, não aleatório como seria com temperature alta (D).]

---
[ ] A - The tool descriptions are ambiguous despite seeming good.
[ ] B - The model is limited and cannot handle this type of request.
[ ] C - The system prompt contains keyword-sensitive instructions creating unintended routing.
[ ] D - The temperature is too high and generates random variation.
