You want Claude to answer medical questions using only sources like nih.gov and PubMed. Which field of the web search tool should you apply?
[Resposta correta: allowed_domains restringe as buscas a domínios específicos, garantindo informação baseada em evidências. max_uses só limita a quantidade de buscas, is_error trata erros de tool_result e input_schema define argumentos, nenhum deles restringe fontes.]

---
[ ] A - max_uses, limiting the number of searches to trusted sources
[ ] B - allowed_domains, restricting the searches to specific domains
[ ] C - is_error, marking blog results as an error
[ ] D - input_schema, filtering by content type
