Which description corresponds to the iterative prompt engineering cycle?
[Resposta correta: O ciclo envolve definir objetivo, escrever, avaliar, aplicar técnicas e re-avaliar iterativamente, com melhora mensurável a cada passo. Usar sem avaliação (A) contraria o ciclo, mexer na temperature (C) e trocar de modelo (D) não são os passos do processo iterativo de engenharia de prompt.]

---
[ ] A - Write a prompt only once and use it in production without evaluation.
[ ] B - Define the goal, write the initial prompt, evaluate against criteria, apply techniques and re-evaluate, repeating the last two steps until performance is satisfactory.
[ ] C - Increase the temperature each round until the output changes.
[ ] D - Swap the model repeatedly until one of them works.
