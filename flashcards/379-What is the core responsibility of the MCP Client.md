What is the core responsibility of the MCP Client?
[Resposta correta: O client é a ponte que abstrai protocolo e mensagens. A é responsabilidade do server. C não é o papel do client. D contorna o server, o que não corresponde ao fluxo real.]

---
[ ] A - Implement the logic of the tools that will be executed.
[ ] B - Serve as the bridge between your server and the MCP servers, abstracting protocol and message exchange.
[ ] C - Store the resources in permanent cache.
[ ] D - Directly call the external service, dispensing with the server.
