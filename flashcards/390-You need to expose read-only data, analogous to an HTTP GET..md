You need to expose read-only data, analogous to an HTTP GET. Which MCP primitive should you use?
[Resposta correta: Resources são o primitivo para buscar informação sem executar ações. A força tools onde resources se aplicam. B confunde com prompts. D não é um primitivo de exposição de dados.]

---
[ ] A - Tools, since only they return data to Claude.
[ ] B - Prompts, because reading data is a user workflow.
[ ] C - Resources, ideal for fetching information rather than executing actions.
[ ] D - Client Session, which reads the data directly.
