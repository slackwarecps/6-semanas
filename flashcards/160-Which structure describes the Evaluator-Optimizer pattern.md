Which structure describes the Evaluator-Optimizer pattern?
[Resposta correta: O padrão combina Producer, Grader e feedback loop. A descreve routing, C descreve paralelização e D descreve um chaining simples sem etapa avaliadora.]

---
[ ] A - A router that classifies the input and picks a pipeline.
[ ] B - A Producer that generates the output, a Grader that evaluates against criteria and a feedback loop that returns feedback to the Producer.
[ ] C - Several parallel requests aggregated by a final step.
[ ] D - A fixed sequence of steps without any evaluation.
