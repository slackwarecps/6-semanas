When is routing the appropriate pattern?
[Resposta correta: Routing serve quando há tipos diversos, categorias claras e categorização confiável. A elimina a razão de rotear. C aponta paralelização e D aponta chaining.]

---
[ ] A - When all requests are identical and use the same approach.
[ ] B - When the app handles diverse types of requests, there are clear categories and the categorization is reliable.
[ ] C - When the sub-tasks are independent and can run in parallel.
[ ] D - When each step depends on the output of the previous step.
