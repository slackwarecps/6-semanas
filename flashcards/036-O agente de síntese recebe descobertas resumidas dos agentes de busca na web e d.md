O agente de síntese recebe descobertas resumidas dos agentes de busca na web e de análise de documentos e, em seguida, passa um resumo consolidado para o gerador de relatórios. Durante os testes, você descobre que os relatórios gerados fazem afirmações factuais sem as devidas citações — o gerador de relatórios não consegue atribuir as declarações às suas fontes originais porque esses metadados foram perdidos durante as etapas de resumo.

Qual é a abordagem mais eficaz para garantir a atribuição correta de fontes nos relatórios finais?

---
[ ] A - Fazer com que cada agente produza dados estruturados que separem os resumos de conteúdo dos metadados de fonte (URLs, nomes de documentos, números de página).
[ ] B - Fazer com que o gerador de relatórios consulte o agente de busca na web para relocalizar as fontes das afirmações no relatório final.
[ ] C - Instruir o agente de síntese a incorporar referências de fonte em linha no texto do resumo usando um formato de citação consistente.
[ ] D - Pular o resumo e passar as saídas brutas completas da busca na web e da análise de documentos diretamente para o gerador de relatórios.
