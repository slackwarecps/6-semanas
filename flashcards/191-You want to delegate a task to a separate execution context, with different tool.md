You want to delegate a task to a separate execution context, with different tooling from the main one. Skill or subagent?
[Resposta correta: O isolamento com tooling próprio é papel do subagent. A e C confundem, pois a skill atua no contexto atual e não isola. D não se aplica ao caso de delegação isolada.]

---
[ ] A - Skill, because it adds expertise to the current context.
[ ] B - Subagent, because it provides isolation and its own context and tooling for the delegated work.
[ ] C - Skill, because it runs in isolation.
[ ] D - Neither, because this requires a routing workflow.
