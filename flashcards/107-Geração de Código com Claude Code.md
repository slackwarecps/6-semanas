Title: Geração de Código com Claude Code
---

Você pediu ao Claude Code para implementar uma função que transforma respostas de API em um formato interno normalizado. Após duas iterações, a estrutura de saída ainda não casa com as expectativas — alguns campos são aninhados de modo diferente e timestamps formatados incorretamente. Você descreveu requisitos em prosa, mas Claude os interpreta de forma diferente a cada vez.

Qual abordagem é mais efetiva para a próxima iteração?

---
[ ] A - Escrever um JSON schema descrevendo a estrutura de saída esperada e validar a saída do Claude contra ele após cada iteração.
[ ] B - Fornecer 2–3 exemplos concretos de entrada-saída mostrando a transformação esperada para respostas representativas da API.
[ ] C - Reescrever requisitos com mais precisão técnica, especificando mapeamentos exatos de campos, regras de aninhamento e strings de formato de timestamp.
[ ] D - Pedir ao Claude para explicar seu entendimento atual dos requisitos para identificar onde as interpretações divergem.
