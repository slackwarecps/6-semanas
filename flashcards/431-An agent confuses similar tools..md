An agent confuses similar tools. What is the first step, low effort and high impact?
[Resposta correta: A descrição é o principal sinal de seleção de ferramenta, então melhorá-la é o primeiro passo de baixo esforço e alto impacto. Camada de roteamento (A), consolidação (C) e few-shot (D) são medidas mais custosas que só devem ser consideradas depois de resolver a descrição.]

---
[ ] A - Build a dedicated routing layer to choose the tool.
[ ] B - Improve the tool descriptions with input formats, examples, edge cases and when to use each one.
[ ] C - Consolidate the similar tools into a single one.
[ ] D - Add many few-shot examples before any other change.
