A cloud company wants to make it easy for agents to use its APIs. Who can author the corresponding MCP server?
[Resposta correta: Qualquer pessoa pode autorar um MCP server, e é comum o provedor do serviço publicar o oficial. A centraliza indevidamente na Anthropic. B restringe demais. D inventa uma exigência de certificação.]

---
[ ] A - Only Anthropic, which controls all MCP servers.
[ ] B - Only the end customer who consumes the service.
[ ] C - Anyone; often the provider itself publishes an official implementation.
[ ] D - Only someone who holds an official MCP certification.
