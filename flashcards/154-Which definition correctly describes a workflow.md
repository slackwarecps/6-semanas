Which definition correctly describes a workflow?
[Resposta correta: Workflow é a sequência pré-determinada de passos para um problema conhecido. A descreve um agent. C descreve o antipadrão de prompt único. D confunde a condição de parada de um loop agêntico e ainda usa critério errado (tokens).]
