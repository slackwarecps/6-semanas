When is it appropriate to consider using extended thinking?
[Resposta correta: Extended thinking é indicado para tarefas complexas quando o prompting padrão falha. Não é necessário em toda tarefa (A), tende a aumentar custo em vez de reduzir (B) e é incompatível com message prefilling (D).]

---
[ ] A - Always, on any task, to guarantee longer responses.
[ ] B - Only when you want to reduce the cost per token.
[ ] C - When standard prompting is not giving good results on hard, complex tasks.
[ ] D - Only together with message prefilling.
