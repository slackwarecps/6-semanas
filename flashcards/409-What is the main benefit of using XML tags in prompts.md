What is the main benefit of using XML tags in prompts?
[Resposta correta: As XML tags funcionam como delimitadores que separam instruções de dados e identificam seções, especialmente com muito conteúdo interpolado. Elas não reduzem tokens (A), não obrigam a saída a ser XML (C) e não têm relação com extended thinking (D).]
