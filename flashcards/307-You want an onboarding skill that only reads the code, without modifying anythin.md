You want an onboarding skill that only reads the code, without modifying anything. How do you configure it?
[Resposta correta: Para restringir escrita, use disallowed-tools (ou deny em /permissions); allowed-tools só pré-aprova, não bloqueia. A A cai na pegadinha do allowed-tools; C e D não impedem edição.]

[ ] B - Use disallowed-tools listing Edit, Write, MultiEdit, or deny rules in /permissions
[ ] C - Omit the frontmatter to disable writing
[ ] D - Set user-invocable: false
