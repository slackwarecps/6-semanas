Title: An agent consistently selects the wrong tool.
---

An agent consistently selects the wrong tool. What is the first diagnostic step?
[Resposta correta: A descrição é o principal input de decisão do modelo, então revisá-la para distinguir propósito e quando usar cada ferramenta é o primeiro passo. Enumerar todos os padrões (A), criar classificador (B) ou remover funcionalidade (D) são abordagens a evitar.]

---
[ ] A - Enumerate all possible usage patterns in the prompt.
[ ] B - Build a separate classifier to choose the tool.
[ ] C - Review the tool descriptions to ensure they clearly distinguish the purpose of each one and when to use them.
[ ] D - Remove the least-used tool to reduce options.
