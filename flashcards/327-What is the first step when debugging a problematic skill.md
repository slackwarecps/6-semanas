What is the first step when debugging a problematic skill?
[Resposta correta: O validator (verifier) é o primeiro passo, pegando problemas estruturais antes de perder tempo. Reinstalar, aumentar effort ou editar o CLAUDE.md não diagnosticam a estrutura da skill.]

---
[ ] A - Reinstall Claude Code
[ ] B - Run the skills validator, which catches structural problems (invalid frontmatter, file name, YAML)
[ ] C - Increase the effort level
[ ] D - Edit the CLAUDE.md to force the skill
