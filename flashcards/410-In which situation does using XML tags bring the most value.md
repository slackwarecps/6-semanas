In which situation does using XML tags bring the most value?
[Resposta correta: As tags ajudam mais quando há muito contexto, mistura de tipos de conteúdo e várias variáveis interpoladas. Em prompts curtos e simples o ganho é mínimo (A), elas não afetam criatividade (B) e não dependem de stop sequences (D).]

---
[ ] A - In short single-sentence prompts with no interpolated data.
[ ] B - When you want the response to be more creative.
[ ] C - When including a lot of context, mixing content types and interpolating multiple variables in complex prompts.
[ ] D - Only when stop sequences are used.
