How do the two types of resource appear in the MCP Inspector?
[Resposta correta: O Inspector separa "Resources" (direct) de "Resource Templates" (templated, que exigem parâmetros). A mistura resources com tools. C omite os templated. D coloca resources na seção errada.]

---
[ ] A - All together in a single list called "Tools".
[ ] B - In separate sections: "Resources" for the direct/static ones and "Resource Templates" for the templated ones, which ask for parameter values.
[ ] C - Only the direct ones appear; the templated ones are not displayed.
[ ] D - Both appear in the "Prompts" section.
