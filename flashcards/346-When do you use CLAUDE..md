When do you use CLAUDE.md versus Skills to organize instructions?
[Resposta correta: CLAUDE.md cobre padrões universais (always-on) e Skills cobrem workflows específicos sob demanda. Mover tudo para Skills faz os padrões universais deixarem de ser aplicados (A); C inverte os papéis; regras por caminho são para convenções por tipo de arquivo, não por workflow (D).]

---
[ ] A - Move everything to Skills, which are more flexible
[ ] B - CLAUDE.md for universal standards (always loaded); Skills for specific workflows, loaded on demand
[ ] C - CLAUDE.md for specific workflows; Skills for universal standards
[ ] D - Use path-based rules for both cases
