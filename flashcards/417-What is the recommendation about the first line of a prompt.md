What is the recommendation about the first line of a prompt?
[Resposta correta: A primeira linha é a mais importante e deve declarar a tarefa de forma direta e clara, como instrução. Perguntas vagas (A) enfraquecem o pedido, exemplos não precedem a instrução principal (B) e stop sequences são parâmetros da API, não conteúdo da primeira linha (D).]

---
[ ] A - It should be an open and vague question to give the model freedom.
[ ] B - It should contain all the few-shot examples before any instruction.
[ ] C - It should lead with a direct, clear statement of the task, in simple language and as an instruction, not as a vague question.
[ ] D - It should bring the list of stop sequences to be used.
