Which sequence describes the typical prompt evaluation workflow?
[Resposta correta: O workflow é rascunhar, criar dataset, rodar, avaliar com grader e iterar comparando ao baseline. Ir direto para produção (B) pula a avaliação, e as opções C e D descrevem técnicas de formatação e geração, não o workflow de avaliação.]

---
[ ] A - Draft the prompt, create an evaluation dataset, run it over each case, pass the output through a grader and change the prompt comparing against the baseline.
[ ] B - Write the prompt and send it straight to production.
[ ] C - Create few-shot examples, increase the temperature and measure tokens.
[ ] D - Define stop sequences, prefix the assistant message and measure latency.
