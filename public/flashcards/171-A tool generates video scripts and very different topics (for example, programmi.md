A tool generates video scripts and very different topics (for example, programming versus surfing) require different styles. Which pattern?
[Resposta correta: O caso pede routing, com categorização do tópico e template especializado. A não se aplica, pois os tópicos são independentes. B desperdiça ao enviar a todos os templates, quando só um é necessário. D adiciona imprevisibilidade sem necessidade.]

---
[ ] A - Chaining, because each topic depends on the previous one.
[ ] B - Parallelization, sending the topic to all templates at the same time.
[ ] C - Routing, categorizing the topic and directing it to a specialized template per category.
[ ] D - Agent, letting Claude pick tools freely.
