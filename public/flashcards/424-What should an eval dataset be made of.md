What should an eval dataset be made of?
[Resposta correta: Um eval dataset contém inputs representativos dos pedidos reais de produção, interpolados no template. Ele não é só outputs (A), não deve conter apenas casos improváveis (C) e não é o código da aplicação (D).]

---
[ ] A - Only the expected outputs, without the inputs.
[ ] B - Example inputs that represent the types of requests the prompt will handle in production, interpolated into the prompt template.
[ ] C - Only extreme cases that never appear in production.
[ ] D - The source code of the application that calls the model.
