The application needs to fetch data to populate an autocomplete in the UI or inject context into the prompt. Which primitive should you use?
[Resposta correta: Resources são controlados pela aplicação, ideais para popular UI ou injetar contexto. A e B têm controladores diferentes. D força uma combinação desnecessária.]

---
[ ] A - Tools, controlled by the model.
[ ] B - Prompts, controlled by the user.
[ ] C - Resources, controlled by the application, which decides when to fetch and how to use (UI or context), as in the "Add from Google Drive" integration.
[ ] D - A tool and a prompt combined.
