You want to test the tools of an MCP server without assembling an entire application. What should you use?
[Resposta correta: O MCP Inspector permite testar as tools sem uma aplicação completa. A e D adicionam trabalho que o Inspector evita. C não é a ferramenta de teste indicada para iterar em tools isoladas.]

---
[ ] A - A separate unit test script for each tool.
[ ] B - The Python SDK's MCP Inspector, started with mcp dev mcp_server.py and accessible via a local URL in the browser.
[ ] C - Claude Code itself in production mode.
[ ] D - A complete MCP Client written from scratch.
