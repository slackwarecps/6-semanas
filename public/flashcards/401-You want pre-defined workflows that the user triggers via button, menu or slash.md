You want pre-defined workflows that the user triggers via button, menu or slash command. Which primitive should you use?
[Resposta correta: Prompts são controlados pelo usuário, que aciona o workflow predefinido. A e B atribuem o controle ao modelo e à aplicação. D não é um primitivo.]

---
[ ] A - Tools, since Claude decides when to run.
[ ] B - Resources, since the application decides when to fetch.
[ ] C - Prompts, controlled by the user, who decides when to run the predefined workflow.
[ ] D - Client Session, which triggers the workflow.
