The user types @document_name to autocomplete and inject the content into the prompt. Which primitive covers this case?
[Resposta correta: Autocomplete e injeção de conteúdo são casos de resources (direct para listar, templated para buscar). A trata leitura de dados como tool. B confunde com prompts. D força tools onde resources são o primitivo adequado.]

---
[ ] A - Tools, since Claude needs to execute an action to fetch the file.
[ ] B - Prompts, because it is the user who triggers the reference.
[ ] C - Resources: a direct one lists the documents (autocomplete) and a templated one fetches the content to inject into the prompt.
[ ] D - A combination of two separate tools, one to list and another to read.
