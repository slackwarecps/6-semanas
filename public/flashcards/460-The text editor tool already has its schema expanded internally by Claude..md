The text editor tool already has its schema expanded internally by Claude. What does the developer still need to provide?
[Resposta correta: Embora Claude expanda o schema a partir de um schema pequeno com type e name, o desenvolvedor ainda precisa escrever a implementação que executa as operações. A opção A ignora o schema embutido, C confunde com a web search tool e D erra ao equiparar à web search, que Claude executa sozinho.]

---
[ ] A - The complete JSON Schema of the arguments, written by hand
[ ] B - The real implementation, that is, the function that executes the operations
[ ] C - The allowed_domains field to restrict access to files
[ ] D - Nothing, since the tool is fully automatic like web search
