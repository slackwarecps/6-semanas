An app needs to handle open-ended requests, whose parameters and the task itself are not known before the user interaction. Which pattern applies?
[Resposta correta: O agent é indicado justamente quando não se sabe de antemão a tarefa nem os parâmetros. A e B forçam um workflow onde a sequência não pode ser pré-determinada. D inverte o tradeoff, pois agents são mais difíceis de testar, não mais fáceis.]

---
[ ] A - Workflow, because every production application should prioritize predictability.
[ ] B - Workflow, because it is possible to pre-determine the sequence of steps.
[ ] C - Agent, because the task and the parameters are unpredictable and require a creative combination of tools.
[ ] D - Agent, because agents are always easier to test.
