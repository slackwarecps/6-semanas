Which command installs the GitHub integration and what does the setup produce?
[Resposta correta: /install-github-app instala o app, adiciona a API key e abre um PR com os workflows (que vão para .github/workflows após o merge). As demais confundem com registro de MCP, /init ou inventam a flag --batch.]

---
[ ] A - claude mcp add github; it generates an MCP server
[ ] B - /install-github-app; it installs the app, adds the API key and generates a PR with the workflow files
[ ] C - /init; it generates a CLAUDE.md with the credentials
[ ] D - claude github --batch; it pushes the workflows straight to main
