Why is it incorrect to state that MCP and tool use are synonyms?
[Resposta correta: MCP entrega a implementação pronta e o tool use é o mecanismo de chamada, sendo os dois complementares. A e C tratam MCP como substituto do tool use. D confunde tool use com resources.]

---
[ ] A - Because tool use does not exist when MCP is used.
[ ] B - Because MCP provides the schemas and functions already implemented, while tool use is how Claude calls those tools; they are complementary.
[ ] C - Because MCP completely replaces tool use in production.
[ ] D - Because tool use is only a subset of resources.
