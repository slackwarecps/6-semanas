You want to expose a capability that only reads data, with no side effect. What should you use?
[Resposta correta: Resource é para buscar dados sem efeito colateral, como um GET. A ignora a distinção tool versus resource. C confunde leitura de dados com workflow de prompt. D é falso, resources também retornam dados.]

---
[ ] A - Tool, since every MCP capability is a tool.
[ ] B - Resource, analogous to an HTTP GET handler that only reads information.
[ ] C - Prompt, because reading data is always a user workflow.
[ ] D - Tool, because only tools can return data to Claude.
