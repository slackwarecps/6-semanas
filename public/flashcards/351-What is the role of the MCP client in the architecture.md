What is the role of the MCP client in the architecture?
[Resposta correta: O client é a ponte que abstrai protocolo e mensagens. A é papel do MCP server, que faz a chamada externa. C é responsabilidade do server ao expor tools. D não descreve o client.]

---
[ ] A - Directly execute the external API and return the result to the user.
[ ] B - Be the communication bridge between your server and the MCP servers, handling message exchange and the protocol.
[ ] C - Define the schemas of the tools that will be exposed to Claude.
[ ] D - Store the resources on disk for offline access.
