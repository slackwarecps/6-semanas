Which description correctly differentiates the two types of resource?
[Resposta correta: Direct tem URI fixa e templated tem parâmetro na URI. A inverte as duas definições. C nega a existência do parâmetro no templated. D inventa uma distinção de leitura versus escrita que não existe.]

---
[ ] A - Direct resource uses a URI with a parameter and templated resource uses a fixed URI.
[ ] B - Direct resource uses a fixed URI (docs://documents) and templated resource uses a URI with a parameter (docs://documents/{doc_id}).
[ ] C - Both always use fixed URIs, changing only the mime_type.
[ ] D - Direct resource is for writing and templated resource is for reading.
