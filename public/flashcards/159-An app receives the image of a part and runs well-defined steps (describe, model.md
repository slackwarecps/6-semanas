An app receives the image of a part and runs well-defined steps (describe, model with CadQuery, render, evaluate and correct) until it generates the STEP file. Which pattern?
[Resposta correta: O fluxo é conhecido e fixo, logo é workflow. A pegadinha de A é assumir que múltiplos passos implicam agent. B e D não se aplicam, pois os passos são sequenciais e dependentes, não independentes nem uma classificação de entrada.]

---
[ ] A - Agent, because there are several steps involved.
[ ] B - Routing, because the input is an image.
[ ] C - Workflow, because the flow is known and can be written in code as a fixed sequence.
[ ] D - Parallelization, because each step is independent.
