What is the robust way to test whether a skill's description triggers correctly?
[Resposta correta: Testar múltiplas variações de frase revela quais formulações não disparam, e as palavras-chave dessas devem entrar na description. Testar só uma frase (A) não cobre a variedade real, a validação de sintaxe (C) não garante disparo e a latência (D) não mede disparo.]

---
[ ] A - Test only one exact phrase and assume the rest will work.
[ ] B - Test phrase variations that express the same request and, if any does not trigger, add those keywords to the description.
[ ] C - Check whether the skill passes syntax validation.
[ ] D - Measure the skill's response time in production.
