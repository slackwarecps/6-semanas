Which decision mapping between MCP primitives is correct?
[Resposta correta: Tools para nova capacidade, Resources para dados na app e Prompts para workflows do usuário. A e D embaralham o mapeamento. C ignora a distinção entre os primitivos.]

---

[ ] B - New capability for Claude (Tools); data for the app, UI/context (Resources); user-triggered workflows (Prompts).
[ ] C - All cases (Tools).
[ ] D - New capability for Claude (Prompts); data for the app (Tools); user workflows (Resources).
