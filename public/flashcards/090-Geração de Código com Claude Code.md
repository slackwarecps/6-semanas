Title: Geração de Código com Claude Code
---

Você quer criar um slash command customizado `/review` que rode o checklist padrão de code review do seu time. Ele deve estar disponível para todo dev quando clonar ou atualizar o repositório.

Onde criar o arquivo do comando?

---
[ ] A - Em `~/.claude/commands/` no diretório home de cada dev.
[ ] B - No repositório do projeto sob `.claude/commands/`.
[ ] C - Em `.claude/config.json` como array de comandos.
[ ] D - No CLAUDE.md raiz do projeto.
