Why does a pure language model not read files on its own and how does tool use solve this?
[Resposta correta: Um language model puro apenas processa e retorna texto; o tool use injeta instruções para o modelo pedir ações, o sistema executa a ação real e devolve o resultado ao modelo. As demais partem da premissa falsa de que o modelo já lê arquivos ou atribuem a causa a rede e memória.]

---
[ ] A - It reads files, but needs network permission; tool use only authenticates
[ ] B - LMs only process text; tool use teaches the model to request actions, executes them and returns the result
[ ] C - LMs do not read because they lack memory; tool use increases the context
[ ] D - LMs read files, but tool use only improves the formatting of the response
