When opening the MCP Inspector, what is the first step before listing or testing tools?
[Resposta correta: É preciso clicar em Connect para inicializar o server antes de listar tools. A, C e D descrevem passos de outros contextos, não o primeiro passo dentro do Inspector.]

---
[ ] A - Write the JSON schema of the tools manually.
[ ] B - Click "Connect" to initialize the server, changing the status from "Disconnected" to "Connected".
[ ] C - Add the server to the project's .mcp.json.
[ ] D - Authenticate via ${GITHUB_TOKEN}.
