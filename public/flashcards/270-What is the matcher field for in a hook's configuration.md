What is the matcher field for in a hook's configuration?
[Resposta correta: O matcher especifica quais tools disparam o hook, aceitando padrões como Write|Edit ou *. As demais descrevem exit code, caminho ou controle de usuário, não a função do matcher.]

---
[ ] A - It defines the success exit code
[ ] B - It specifies which tool types trigger the hook (e.g. Read, Write|Edit, or *)
[ ] C - It defines the absolute path of the script
[ ] D - It restricts which users can run the hook
