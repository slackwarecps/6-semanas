You are going to continue a related task and want to preserve what Claude learned. Which command?
[Resposta correta: /compact resume o histórico e preserva o aprendizado, ideal para tarefas relacionadas. /clear zeraria o contexto (para tarefa diferente); /rewind volta a um ponto; /resume não inicia contexto novo.]

---
[ ] A - /clear, since it wipes the context
[ ] B - /compact, since it summarizes the history preserving the learning
[ ] C - /rewind, since it goes back to an earlier point
[ ] D - /resume, since it starts a new context
