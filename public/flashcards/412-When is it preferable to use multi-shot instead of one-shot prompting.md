When is it preferable to use multi-shot instead of one-shot prompting?
[Resposta correta: Multi-shot é indicado para cobrir múltiplos edge cases e diferentes formatos válidos de resposta. Ele aumenta tokens em vez de reduzir (A), não é necessário quando um exemplo basta (B) e não tem relação com temperature (D).]

---
[ ] A - When you want to reduce the number of tokens in the prompt as much as possible.
[ ] B - When a single example already covers all the cases.
[ ] C - When you need to cover several edge cases or show different types of valid response.
[ ] D - When the goal is to increase the temperature of the response.
