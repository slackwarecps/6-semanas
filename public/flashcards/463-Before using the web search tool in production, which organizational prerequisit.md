Before using the web search tool in production, which organizational prerequisite must be met?
[Resposta correta: A organização precisa habilitar a Web Search tool no console de settings antes do uso. allowed_domains é configuração da chamada, fine_grained é recurso não relacionado e a busca não é implementada pelo desenvolvedor.]

---
[ ] A - Define allowed_domains for all users
[ ] B - Enable the Web Search tool in the organization settings console
[ ] C - Enable fine_grained tool calling
[ ] D - Manually implement the search function
