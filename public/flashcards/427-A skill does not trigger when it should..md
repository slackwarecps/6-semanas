A skill does not trigger when it should. What is the indicated adjustment?
[Resposta correta: O ajuste é enriquecer a description com palavras-chave e frases de gatilho alinhadas ao modo como os pedidos são feitos, pois o matching é semântico. Mexer no código interno (B), no effort (C) ou reduzir exemplos (D) não afeta o disparo da skill.]

---
[ ] A - Add to the description more keywords and trigger phrases that match how you actually phrase your requests.
[ ] B - Rewrite the skill's internal code so it runs faster.
[ ] C - Increase the session effort so the model looks for more skills.
[ ] D - Reduce the number of examples in the main prompt.
