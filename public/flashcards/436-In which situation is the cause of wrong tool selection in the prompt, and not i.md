In which situation is the cause of wrong tool selection in the prompt, and not in the tool description?
[Resposta correta: Um padrão sistemático ligado a uma palavra-chave, com acerto na ausência dela, aponta para lógica keyword-sensitive no system prompt. Erro aleatório (A) não indica roteamento por termo, descrições parecidas (C) apontam para a descrição e não o prompt, e o acerto sem o termo descarta limitação do modelo (D).]

---
[ ] A - When the error occurs completely at random, unrelated to the text of the request.
[ ] B - When there is a systematic pattern triggered by a specific keyword and the model gets it right in the absence of that term, indicating keyword-sensitive logic in the system prompt.
[ ] C - When the tool descriptions are too similar to each other.
[ ] D - When the model is too small for the task.
