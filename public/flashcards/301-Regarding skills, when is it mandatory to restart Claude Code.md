Regarding skills, when is it mandatory to restart Claude Code?
[Resposta correta: O restart só é necessário ao criar um diretório top-level de skills que não existia no início da sessão. Edições e remoções entram automaticamente, e o item D exagera ao dizer "nunca".]

---
[ ] A - Whenever any SKILL.md is edited
[ ] B - Only when creating a top-level skills directory that did not exist when the session started
[ ] C - Whenever a skill is removed
[ ] D - Never, all changes are automatic
