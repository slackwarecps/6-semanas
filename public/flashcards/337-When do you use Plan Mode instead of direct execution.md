When do you use Plan Mode instead of direct execution?
[Resposta correta: Plan Mode é para tarefas complexas, de larga escala ou com decisões arquiteturais, onde trade-offs surgem na exploração. Execução direta cobre tarefas simples e bem definidas, então A e D estão erradas; testes não são o critério (C).]

---
[ ] A - On any task, even simple and well-defined ones
[ ] B - When the task is complex, large-scale, with multiple valid approaches or architectural decisions
[ ] C - Only when you need to run tests
[ ] D - Only for one-line tasks
